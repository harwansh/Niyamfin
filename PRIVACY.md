# Privacy Notice

Niyamfin is designed as a browser-first financial-health calculator.

## App storage

The app code does not use a database, cookies, localStorage, sessionStorage, or analytics. Financial inputs are held in React component state while the page is open and are cleared when the page is refreshed or closed.

## Network behavior

The deterministic financial report is computed locally from the values entered in the form. There is no server-side summary route in the current codebase.

## External resources

Avoid hard-coded third-party scripts, trackers, and analytics. Prefer framework-managed or self-hosted static assets. Document any external request here before release.

## Sensitive data guidance

Use approximate household financial amounts. Do not enter account credentials or official identification numbers.

## Disclaimer

The report is educational and should not be treated as financial, investment, insurance, tax, or legal advice. Verify important decisions with a qualified professional.
